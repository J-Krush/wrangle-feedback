# Wrangle Feedback

[Wrangle](https://wrangle.app) is a native macOS menu bar app that gives you quick access to your pull requests across GitHub repositories.

This is the **public feedback repository** for Wrangle. The source code is not available here — this repo exists solely for bug reports and feature requests.

## Download

Get Wrangle at [wrangle.app](https://wrangle.app).

## Reporting Bugs

Found a bug? [Open a bug report](../../issues/new?template=bug_report.md). Please include:

- Your Wrangle version
- Your macOS version
- Steps to reproduce the issue

## Requesting Features

Have an idea? [Open a feature request](../../issues/new?template=feature_request.md). Describe the feature and why it would be useful to you.
